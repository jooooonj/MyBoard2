# SpringBoot-Project- Q&A 웹게시판
스프링 부트로 만드는 Q&A 웹게시판 서비스


## 🖥️ 프로젝트 소개
자유로이 질문과 답변을 주고 받을 수 있는 웹게시판 서비스

## 🕰️ 개발 기간
* 23.03.30일 - 23.03.31일

### ⚙️ 개발 환경
- `Java 17`
- `JDK 1.8.0`
- **IDE** : IntelliJ
- **Framework** : Springboot
- **Database** : MySQL
- **ORM** : Spring Data JPA
